# Never Stop Blowing Up Companion

An interactive rules companion for **Never Stop Blowing Up** inside TaleSpire.

Based on the TTRPG system by Brennan Lee Mulligan & Carlos Luna (Dropout), itself based on Kids on Bikes/Kids on Brooms.

## Features

- **Character Sheet**: Track your 9 skills (Stunts, Brawl, Tech, Weapons, Sneak, Wits, Tough, Drive, Hot) as die types from d4 to d20. Manage Turbo Tokens, Injury Levels, and owned Abilities. Auto-saved per campaign.
- **Dice Roller**: Roll skill checks wired to TaleSpire's dice tray. Detects Blow Ups (rolling max on a die) and automatically upgrades your skill die. Includes Prepared Action quick-reference and a roll log.
- **Rules Reference**: Searchable, collapsible reference for all game mechanics -- Blowing Up, Turbo Tokens, Injury Levels, Abilities, Group Suites, and more.

## How to Use

1. Open the Symbiote from TaleSpire's side panel.
2. Use the **tabs** at the top to switch between Sheet, Dice, and Rules.
3. On the **Sheet** tab, set your skill die types by clicking the small buttons under each skill. Click a skill box to roll it.
4. On the **Dice** tab, click any skill button to roll that skill check. If you Blow Up, the skill die is automatically upgraded!
5. Track **Turbo Tokens** with the +/- buttons or the quick-add buttons (Failed Check, Adrenalized).
6. Click **Injury Levels** to set your current injury state.
7. Check off **Abilities** you own on the character sheet.

## The Blow Up Mechanic

When you roll the maximum value on your skill die, it **Blows Up**:
- Your skill permanently upgrades to the next die type
- You should roll the new die and add the result to your total
- If the new roll is also the max, keep Blowing Up!

The Symbiote automatically upgrades your skill die on a Blow Up. For chain rolls, click the skill again on the Dice tab to roll the upgraded die.
